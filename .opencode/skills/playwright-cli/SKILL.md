# Playwright CLI Browser Automation

Use this skill when you need to navigate websites, interact with web pages, fill out forms, take screenshots, or extract information from web applications using `playwright-cli` directly via the terminal.

## Command Discovery & Syntax Verification

Because `playwright-cli` is pre-installed globally in this environment, you have direct access to its internal manual.

**If you are unsure of a command's syntax, flags, or subcommands, you must run:**

```bash
playwright-cli --help
```
